# microinteracoes

A React Native Expo project showcasing a collection of **gesture-based micro-interactions** using `react-native-gesture-handler` and `react-native-reanimated`.

## Objective

The objective of this project is to explore and demonstrate different types of touch gestures and animations on mobile, serving as a reference for building polished, interactive UI experiences in React Native.

## Gestures Demonstrated

| Component | Gesture |
|---|---|
| `Fling` | Quick swipe/fling gesture |
| `LongPress` | Sustained press interaction |
| `Pan` | Drag along a single axis |
| `PanXY` | Free 2D drag |
| `Pinch` | Two-finger zoom |
| `Rotation` | Two-finger rotation |
| `Touches` | Raw touch tracking |

## Tech Stack

- React Native
- Expo
- TypeScript
- react-native-gesture-handler
- react-native-reanimated

## Getting Started

```bash
npm install
npx expo start
```
